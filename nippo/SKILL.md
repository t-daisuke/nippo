# 日報生成スキル

以下のセッションデータから日報を作成してください。

## セッションデータ

!`~/.claude/skills/nippo/scripts/collect-logs.sh $ARGUMENTS`

## 出力形式

上記データを分析し、以下の形式で日報を作成：

```markdown
# 日報 - YYYY-MM-DD

## 今日やったこと

### [プロジェクト名]
- やったこと（ユーザーメッセージから推測）

## 詳細（任意）

セッションごとの詳細が必要な場合のみ記載

## 明日やること（継続中のタスクがあれば）
- タスクリスト
```

### 注意

- ユーザーメッセージから作業内容を推測してまとめる
- 機密情報（パスワード、トークン等）は日報に含めない
- 同じプロジェクトの作業はグルーピング

## 引数

- `/nippo` - 今日の日報
- `/nippo yesterday` - 昨日の日報
- `/nippo 2026-01-20` - 指定日の日報
